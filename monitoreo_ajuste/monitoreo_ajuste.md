# Monitoreo y Ajuste

## Método de Monitoreo
- Usar Google Docs para rastrear el progreso de cada tarea.
- Actualizar el documento semanalmente con el estado de cada tarea.

## KPIs
- Porcentaje de tareas completadas.
- Tiempo promedio para completar una tarea.

## Estrategias de Ajuste
- Si el progreso no cumple con las expectativas, reevaluar las prioridades y deadlines.
- Considerar reasignar tareas o recursos para acelerar el progreso.
