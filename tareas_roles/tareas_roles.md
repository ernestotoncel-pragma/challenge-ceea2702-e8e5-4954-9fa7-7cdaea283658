# Tareas y Roles

## Tareas
- Tarea 1: Reunirse con el cliente para definir los requisitos.
- Tarea 2: Diseñar la arquitectura del proyecto.
- Tarea 3: Desarrollar las funcionalidades principales.
- Tarea 4: Realizar pruebas de usuario.

## Roles
- Rol 1: Gerente de Proyecto - Responsable de coordinar las tareas y asegurar que el proyecto se complete a tiempo.
- Rol 2: Desarrollador - Responsable de implementar las funcionalidades del proyecto.
- Rol 3: Diseñador - Responsable de crear el diseño de la interfaz de usuario.
- Rol 4: Probador - Responsable de realizar pruebas y asegurar que el producto final cumpla con los requisitos.
